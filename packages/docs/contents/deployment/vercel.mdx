# Vercel Deployment

The gallery can be deployed to Vercel as a static Vite application.

## Project Settings

Use these settings:

```plain
Framework Preset: Vite
Install Command: pnpm install
Build Command: pnpm run build:manifest && SKIP_PRECHECK=1 pnpm build
Output Directory: apps/web/dist
Node.js Version: 24
```

`SKIP_PRECHECK=1` is used because the manifest has already been generated by the first build command.

## Notes

- Keep `config.json.url` aligned with the production Vercel or custom domain.
- The current repository workflow also commits `web/`; Vercel should build from `apps/web/dist` unless you intentionally configure it to serve the committed `web/` directory.
- If using the current local storage setup, make sure original image URLs in `baseUrl` are publicly reachable by the deployed site.
